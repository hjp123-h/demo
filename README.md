## 快速运行
1. 安装必备工具  
````
JDK，Maven
````
2. 克隆代码到本地
````
git clone
````

3. 修改数据库配置文件
````

````
4. 运行打包命令
````
mvn package
````
5. 运行项目  
````
java -jar /user/project/demo.jar
````
6. 访问项目
````
http://localhost:8088
````
